Student Exam Performance Indicator

A Machine Learning Project to Predict Student Math Scores

Project Overview

This project predicts the math scores of students based on various demographic and academic features. It helps understand how factors like gender, ethnicity, parental education, lunch type, and test preparation affect student performance. The project includes data analysis, model training, evaluation, and deployment as a web application.

Features Used

  • Gender
  • Race/Ethnicity
  • Parental Level of Education
  • Lunch
  • Test Preparation Course
  • Reading Score
  • Writing Score

Technologies and Libraries Used

Python pandas numpy scikit-learn CatBoost XGBoost Flask Bootstrap Jupyter Notebook

Workflow

1. Data Collection & Exploration
  • Collects student data: gender, ethnicity, parental education, lunch type, test preparation, reading & writing scores.
  • Performs EDA to understand data distribution and relationships.
2. Data Preprocessing
  • Handles missing values and encodes categorical features using OneHotEncoder.
  • Scales numerical features using StandardScaler.
  • Splits data into training and testing sets.
3. Model Training & Evaluation
  • Trains multiple regression models: Linear, Ridge, Lasso, KNN, Decision Tree, Random Forest, XGBoost, CatBoost, AdaBoost.
  • Evaluates models using MAE, RMSE, and R2 Score.
  • Selects the best performing model for deployment.
4. Model Deployment
  • Deploys the model using a Flask web application.
  • Users can input student data and get predicted math scores instantly.
5. User Interface
  • Modern, responsive UI built with Bootstrap.
  • Easy navigation and clear instructions for users.

Model Evaluation Metrics

  • Mean Absolute Error (MAE)
  • Root Mean Squared Error (RMSE)
  • R2 Score

How to Use

  1. Click the Prediction Calculator button in the navigation bar or below.
  2. Fill in the student details and scores on the next page.
  3. Submit the form to get an instant math score prediction.
